Etiquetas Label e Input

La etiqueta <label> se utiliza para definir un formulario en concreto, vinculándolo y dejando al usuario interacturar con este. Cada etiqueta <label> está vinculada a un <input> a través del atributo for. La etiqueta <input> se utiliza para crear los elementos interactivos de los formularios.

Es recomendable, a nivel de sintaxis, colocar el nombre del campo del formulario en un span.

<form action="">


<label for="nombre">

<span">Nombre: </span >

<input type="text" name="nombre">

</label>


</form>

También es importante saber que la etiqueta input posee varios atributos, entre ellos encontraremos:

type

Indica el tipo de control que vamos a mostrar, por defecto se le asigna el valor "text". Entre los valores que puede tener se encuentran:

  • text: crea un campo de texto simple, es el valor por defecto.

  • password: permite introducir una cadena de manera oculta. Se le puede añadir el atributo "maxlenght" para definir una longitud máxima.

  • email: crea un campo en el que tan solo podremos introducir una dirección de correo electrónico.

  • tel: permite introducir un número de teléfono. Podemos incluir los atributos "pattern" y "maxlenght" para restringir los valores introducidos.

  • url: permite introducir una URL.

  • number: tan solo permite introducir números.

  • color: permite especificar un color.

  • file: permite al usuario seleccionar un archivo. Podemos añadirle en atributo "accept" para indicar los tipos de archivos que podrá seleccionar.

  • image: permite al usuario enviar una imagen. Podemos añadir el atributo "alt" para definir un texto alternativo, y height y width para definir el tamaño de la imagen.

  • date: permite introducir una fecha (año, mes y día).

  • datetime: permite introducir una fecha y, además, una hora (horas, minutos, segundos y fracciones de segundo).

  • month: permite introducir un mes y año.

  • week: permite introducir una fecha, más concretamente una semana del año.

  • radio: crea una casilla de selección, en caso de haber varias tan solo podremos seleccionar uno. Con el atributo "value" indicamos el valor que se enviará por ese elemento.

  • checkbox: crea una casilla de selección, en caso de haber varias podemos seleccionar todas las que queramos. Con el atributo "value" indicaremos el tipo de valor que se enviará por elemento.

  • button: crea un botón sin un comportamiento asignado.

  • submit: crea botón para enviar el formulario.

  • reset: crea un botón que restaura todos los contenidos del formulario a sus valores predeterminados.

placeholder

Toma un valor de ejemplo para mostrar en el campo de entrada cuando esté vacío.

Etiqueta Textarea

Representa un control de escritura de texto de manera multilínea. Es recomendable indicar el ancho de la caja con el atributo "cols", en caso de no hacerlo su valor predeterminado será de 20. De la misma forma podemos definir la altura con "rows"

<form action="">


<textarea name="comentario" id="comentario cols="30" rows="10">


</form>

Al igual que con cualquier otra etiqueta, disponemos de diversos atributos, entre ellos:

readonly

Indica que el usuario no puede modificar el contenido, pero si podrá seleccionarlo.

resize

Nos permite definir el comportamiento que tomará el <textarea> cuando intentemos modificar su tamaño como usuarios. Este no es un atributo realmente de la etiqueta HTML, sino una propiedad del CSS.

  • none: no permite al usuario redimensionarlo.

  • both: es dimensionable en ambos ejes.

  • horizontal: es dimensionable solamente en el eje horizontal.

  • vertical: es dimensionable solamente en el eje vertical.
