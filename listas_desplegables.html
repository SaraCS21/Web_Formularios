Etiquetas Select y Datalist

Para crear inputs con una lista de varias opciones, se puede desarrollar de dos maneras diferentes:

  • Etiqueta <select>: esta permite crear la lista, con las etiquetas <option>:


    • <select name="clases">


    <option name="DSW">Desarrollo web en entorno servidor </span >

    <option name="DOR">Diseño de interfaces web </span >

    <option name="DPL">Despliegue de aplicaciones web </span >

    <option name="DEW">Desarrollo web en entorno cliente </span >


    </select>

  • Etiqueta <input list = "" >: de este modo, se puede utilizar una etiqueta <datalist> con etiquetas <option> dentro del input. De este modo, el usuario puede escribir dentro del input, y filtrar los resultados de la lista (mejor opción):


    • <input list="clases">

    <datalist id="clases">


    <option name="DSW">:Desarrollo web en entorno servidor </span >

    <option name="DOR">:Diseño de interfaces web </span >

    <option name="DPL">:Despliegue de aplicaciones web </span >

    <option name="DEW">:Desarrollo web en entorno cliente </span >


    </datalist>

Etiquetas Option y Optgroup

<option> es usado para insertar opciones en una lista de opciones (<select>). Estos pueden ser agrupados en diferentes grupos con <optgroup>

<select name="clases">


<optgroup label="Primero de DAW">

<option name="PRO">Programación </span >

<option name="LND">Lenguaje de marcas </span >

<option name="BAE">Base de datos </span >

<option name="ETS">Entornos de desarrollo </span >

<option name="SSF">Sistemas informáticos </span >

</optgroup>


<optgroup label="Segundo de DAW">

<option name="DSW">Desarrollo web en entorno servidor </span >

<option name="DOR">Diseño de interfaces web </span >

<option name="DPL">Despliegue de aplicaciones web </span >

<option name="DEW">Desarrollo web en entorno cliente </span >

</optgroup>


</select>
